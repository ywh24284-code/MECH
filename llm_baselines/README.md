# LLM Baselines

This folder contains LLM fine-tuning baselines.

## Files

- `train_llm_qlora.py`: QLoRA training script
- `train_llm_qlora_v2.py`: Updated QLoRA training pipeline
- `compare_llm_baselines.py`: Compare LLM baseline outputs
- `requirements.txt`: LLM-specific dependencies

## Example

```bash
python llm_baselines/train_llm_qlora.py --model_path meta-llama/Meta-Llama-3.1-8B-Instruct
```
