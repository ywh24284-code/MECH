# Prompting Baselines

This folder contains prompt-based inference and evaluation baselines.

## Files

- `run_prompting_baseline.py`: Main prompting baseline runner
- `run_prompting_v2.py`: Updated prompting runner
- `classify_opinions0.py`: Opinion classification helper
- `compare_prompting_baselines.py`: Comparison utility

## Example

```bash
python prompting_baselines/run_prompting_baseline.py
```
